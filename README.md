# progettoClips
Progetto CLIPS per IA Lab 2022

## Clips

Per runnare il codice caricare il file su CLIPS tramite il comando (oppure da interfaccia):
```
(load "case.clp")
```

Eseguire un reset:
```
(reset)
```

Ed infine eseguire:
```
(run)
```
